# Wildberries Parser

## Функционал

Парсер собирает каталог товаров с сайта Wildberries по поисковому запросу и сохраняет данные в XLSX.

Собираемые данные:

- ссылка на товар
- артикул
- название
- цена
- описание
- ссылки на изображения
- характеристики
- название селлера
- ссылка на селлера
- размеры
- остатки
- рейтинг
- количество отзывов

## Фильтрация

Дополнительно создаётся файл с товарами:

- рейтинг >= 4.5
- цена <= 10000
- страна производства = Россия

## Установка
```bash
pip install -r requirements.txt

## Запуск
python parser.py --query "пальто из натуральной шерсти" --pages 3 --out output

## Результат

Создаются файлы:
output/full_catalog.xlsx
output/filtered_catalog.xlsx

## Примечание

Парсер использует открытые API Wildberries и делает паузы между запросами,
чтобы снизить нагрузку на сервер.
